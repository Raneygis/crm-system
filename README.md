# ?? CRM System - Django REST Framework

Полнофункциональная CRM система на Django с REST API.

## ? Возможности
- ?? Управление контактами
- ?? Воронка продаж
- ? Задачи и активности
- ?? Пользователи и роли
- ?? JWT аутентификация

## ?? Быстрый старт
```bash
git clone https://github.com/Raneygis/crm-system.git
cd crm-system
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt
python manage.py migrate
python manage.py runserver
```
